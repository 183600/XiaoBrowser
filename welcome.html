权限

contextMenus: This to add all the Page Sidebar item in the web browser context menu.
storage: This to save the settings on your local computer, and to sync it to your web browser account.
declarativeNetRequestWithHostAccess: This is to load any website into the side panel.
sidePanel: This is to show the side panel in the web browser.
上下文菜单：用于在网页浏览器的上下文菜单中添加所有页面边栏项。
存储：用于在本地计算机上保存设置，并将其同步到您的网页浏览器账户。
具有主机访问权的声明性网络请求：用于将任何网站加载到侧边栏中。
侧边栏：用于在网页浏览器中显示侧边栏。
